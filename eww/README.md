## EWW

### Prerequisites

EWW: https://github.com/elkowar/eww

(MORE ???)

### Setup

0. Make sure you have a `$HOME/.config/eww` folder & you have run `eww daemon`
1. Run `deploy`
2. Open an eww window with `eww open <window>` (or run `trigger`)

### Development

1. Make some changes to files under `src`
2. Run `deploy`
3. Profit
